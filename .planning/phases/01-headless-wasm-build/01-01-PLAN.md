<objective>
Establish the three-stage Emscripten cross-compilation build system for Blender WASM.

Purpose: Nothing else in Phase 1 can proceed without a working build pipeline. This plan creates the Dockerfile for reproducible builds, the CMake configuration that disables unsupported features and enables WASM-specific flags (SIMD, pthreads, mimalloc, PROXY_TO_PTHREAD), the native host tools build (Stage 1), the dependency resolution (Stage 2), and the WASM build orchestration script (Stage 3).

Output: Working build infrastructure that can configure and begin compiling Blender to WASM.
</objective>

         -sSTACK_SIZE=2MB
